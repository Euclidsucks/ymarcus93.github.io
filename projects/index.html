Main Projects

Bitcoin Profit Calculator

This is a side project I created a year ago when I was first learning Java. The project is supposed to help the user calculate their profits using Bitcoin. The program stores the amount of Bitcoin the user purchased, how much the user paid for it, and when the user paid for it in a database. Then, the program retrieves the latest Bitcoin price and calculates profits or losses.

You can see the source code here on GitHub.

Sephardi Jews iPhone App

This was an app I created three years ago when I was first learning Objective-C and iOS development. It is an app that connects to an old website I created that discusses the customs of Sephardic Jews. The app connects to the RSS feed of the website and pulls it down for the user. In additon, the app includes other interactive features and useful resources for those who want to learn more about Sephardi Jews.

You can download the app here.
